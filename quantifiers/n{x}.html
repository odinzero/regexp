RegExp {x} Quantifier

Definition and Usage

The n{x} quantifier matches any string that contains a sequence of X n's.

X must be a number.

Browser Support



The {x} quantifier is supported in all major browsers.

Syntax

Syntax with modifiers

new RegExp("n{X}")

or simply:

/n{X}/
new RegExp("n{X}", "g")

or simply:

/\n{X}/g
